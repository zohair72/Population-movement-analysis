# Web GIS Dashboard

A full-stack Web GIS Dashboard application for geospatial data visualization and analysis.

## Tech Stack

| Layer      | Technologies                          |
|------------|---------------------------------------|
| Frontend   | React, Tailwind CSS, Mapbox, Deck.gl  |
| Backend    | Node.js, Express                      |
| Database   | MongoDB                               |

## Project Structure

```
web-gis-dashboard/
├── frontend/    # React frontend with Mapbox & Deck.gl map components
├── backend/     # Node.js/Express REST API server
├── database/    # MongoDB configuration, schemas & seed data
├── docs/        # Project documentation and resources
└── README.md    # This file
```

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn
- MongoDB (local or Atlas)
- Mapbox API Token

### Installation

1. Clone the repository
2. Install frontend dependencies: `cd frontend && npm install`
3. Install backend dependencies: `cd backend && npm install`
4. Configure environment variables (see `.env.example` files)
5. Start the development servers
